These are compiled worldmap and streets sidecars generated by the Dru_map
authoring pipeline. PZMapForge has `worldmap.xml` and `worldmap-forest.xml`
(uncompiled stubs) but does not have the compiled `.bin` versions or the
`.bak` variants.

Whether any of these files are required for IsoMetaGrid to mount the map
folder is currently an open hypothesis. They may be:
- Required for IsoMetaGrid map-folder registration.
- Required for in-game world-map rendering only (not registration).
- Optional/informational.

The next probe should determine which, if any, of these files are needed
for runtime mounting.
